# Controle Fácil – PWA (v0.1)

## Como rodar (modo teste)
1) Abra um terminal na pasta do projeto
2) Rode um servidor local (PWA exige HTTPS ou localhost):
   - Python: `python -m http.server 8080`
3) Abra no navegador: http://localhost:8080

## Como instalar no celular
- No Chrome/Edge (Android): menu ⋮ -> "Instalar app"
- No desktop: ícone de instalar na barra de endereços

## Observação
- Este v0.1 já tem: Home com botões, Venda com leitor/câmera (BarcodeDetector), Estoque, Contas, Fiado, Relatórios CSV, Dashboard básico e Admin para propagandas.
